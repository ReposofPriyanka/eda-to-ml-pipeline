# eda-to-ml-pipeline
End-to-end data science pipeline combining exploratory data analysis, visualization, and machine learning to uncover patterns, generate insights, and build predictive models on real-world data.

This folder contains six EDA projects completed using Python in Visual Studio Code (VS Code).

Each project includes:
- Cleaned datasets (`data/`)
- Analytical outputs (`outputs/`)
- Visualizations (`plots/`)
- Python scripts (`scripts/`)

🔍 Projects Descriptions:
1. Customer Segmentation Analysis 
   Performed K-Means clustering to segment mall customers based on age, income, and spending score.  
   Includes visualizations and insights on customer behavior.

2. COVID-19 Dataset EDA
   Analyzed global COVID-19 data to explore case trends, death rates, and country-wise impact.  
   Focused on data cleaning, time series trends, and correlation analysis.

3. IRIS Dataset EDA
   Explored the classic IRIS dataset to visualize relationships between sepal and petal dimensions.  
   Includes pairplots, correlation matrices, and class distribution analysis.

4. EDA on Movie Rating Dataset 
   Analyzed movie rating patterns across genres, popularity, and viewer preferences.  
   Derived insights on rating distributions, correlations, and trends.

5. EDA on Stock Market Data  
   Used `yfinance` and `mplfinance` to fetch and visualize stock data.  
   Conducted trend analysis, correlation study, and candlestick chart visualization.

6. EDA on Weather Data
   Analyzed temperature, humidity, and rainfall patterns across regions.  
   Visualized seasonal variations and correlations between meteorological parameters.
